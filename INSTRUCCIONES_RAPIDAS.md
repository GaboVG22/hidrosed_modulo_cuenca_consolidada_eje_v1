# Instrucciones rápidas

1. Ejecutar:
   ```bash
   pip install -r requirements.txt
   streamlit run app.py
   ```

2. Cargar `Cuenca consolidada.kmz`.

3. Ajustar:
   - buffer/corredor a cada lado del eje;
   - separación de transectas;
   - equidistancia de curvas auxiliares;
   - caudal incremental si corresponde.

4. Descargar:
   - KMZ preparado;
   - Excel de resultados;
   - ZIP completo.
